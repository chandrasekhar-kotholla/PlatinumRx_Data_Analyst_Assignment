# Data Analyst Assignment

This project demonstrates data analysis using SQL, Excel, and Python on hotel and clinic datasets.

## SQL

* Performed joins, aggregations, and ranking queries
* Solved business problems like revenue analysis, billing, and customer insights

## Excel

* Used INDEX-MATCH for lookups
* Performed date and time-based analysis (same day/hour)

## Python

* Converted minutes into readable time format
* Removed duplicate characters from strings

## Structure

* SQL → schema and query files
* Excel → data analysis workbook
* Python → utility scripts

This project showcases practical data analysis skills and problem-solving using structured data.
